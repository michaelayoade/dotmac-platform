# DotMac Platform (Control Plane)

Central management system for multi-tenant ISP deployments. Manages licensing, tenant provisioning, and partner relationships.

## Installation

```bash
poetry install
```

## Development

```bash
# Run tests
poetry run pytest

# Type checking
poetry run mypy src/

# Linting
poetry run ruff check src/
```
